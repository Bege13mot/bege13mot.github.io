Агрессия. Конрад Лоренц

Mar 4, 2014 · 1 minute read · Comments

Post Tag: biologybook

Удивительная книга. По идее, записи нобелевского лауреата о внутривидовой агрессии. А по сути - разговор с разносторонним, интеллигентным и мудрым автором. Научные факты ни сколько не заслоняют рассуждения.

  • Одна из самых сильных агрессий внутривидовая. Потому что она разносит особей по территории. Выявляет сильных самцов для размножения.

  • Агрессию периодически нужно реализовывать. “Полярное бешенство”. Выйти и громко что-нибудь расколошматить.

  • Без агрессии не может быть любви и дружбы. Смех - это тоже проявление агрессии. Агрессия должна находить выход.

  • Чем выше агрессивность вида, тем сильнее механизмы выключения агрессии. Например волки некогда не нападают на молодняк и самок.
